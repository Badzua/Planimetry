Определение геометрических фигур

Основные фигуры, которые мы будем изучать

Круг

Множество всех точек плоскости, равноудалённых от одной точки (центра).

Квадрат

Прямоугольник, у которого все стороны равны и все углы прямые.

Треугольник

Фигура, образованная тремя отрезками, соединяющими три точки, не лежащие на одной прямой.

Параллелограмм

Четырёхугольник, у которого противоположные стороны параллельны.

Трапеция

Четырёхугольник, у которого две стороны параллельны, а две другие — нет.

Ромб

Параллелограмм, у которого все стороны равны.

Формулы периметра

Периметр — сумма длин всех сторон фигуры

Периметр круга (длина окружности)

Формула: P = 2πr = πd

где r — радиус, d — диаметр, π ≈ 3.14159

Пример: Круг с радиусом 5 см

P = 2 × π × 5 = 10π ≈ 31.42 см

Периметр квадрата

Формула: P = 4a

где a — длина стороны квадрата

Пример: Квадрат со стороной 6 см

P = 4 × 6 = 24 см

Периметр треугольника

Формула: P = a + b + c

где a, b, c — длины сторон треугольника

Пример: Треугольник со сторонами 3, 4, 5 см

P = 3 + 4 + 5 = 12 см

Периметр параллелограмма

Формула: P = 2(a + b)

где a и b — длины смежных сторон

Пример: Параллелограмм со сторонами 5 и 7 см

P = 2 × (5 + 7) = 24 см

Периметр трапеции

Формула: P = a + b + c + d

где a, b — основания, c, d — боковые стороны

Пример: Трапеция со сторонами 4, 6, 3, 5 см

P = 4 + 6 + 3 + 5 = 18 см

Периметр ромба

Формула: P = 4a

где a — длина стороны ромба

Пример: Ромб со стороной 8 см

P = 4 × 8 = 32 см

Формулы площади фигур

Площадь — мера поверхности, ограниченной замкнутой фигурой

Площадь круга

Формулы:

1. S = πr² (через радиус)

2. S = (P²)/(4π) (через периметр)

Пример: Круг с радиусом 3 см

S = π × 3² = 9π ≈ 28.27 см²

Площадь квадрата

Формулы:

1. S = a² (через сторону)

2. S = d²/2 (через диагональ)

Пример: Квадрат со стороной 5 см

S = 5² = 25 см²

Площадь треугольника

Формулы:

1. S = ½ × a × h (через основание и высоту)

2. S = ½ × a × b × sin(γ) (через 2 стороны и угол)

3. S = √[p(p-a)(p-b)(p-c)] (формула Герона)

где p = (a+b+c)/2 — полупериметр

Площадь параллелограмма

Формулы:

1. S = a × h (через основание и высоту)

2. S = a × b × sin(α) (через 2 стороны и угол)

Площадь трапеции

Формулы:

1. S = ½ × (a + b) × h (через основания и высоту)

2. S = m × h (через среднюю линию и высоту)

где m = (a + b)/2 — средняя линия

Площадь ромба

Формулы:

1. S = a × h (через сторону и высоту)

2. S = a² × sin(α) (через сторону и угол)

3. S = ½ × d₁ × d₂ (через диагонали)

JavaScript: Основные функции проверки

// Проверка существования треугольника по неравенству треугольника
function isValidTriangle(a, b, c) {
return a + b > c && a + c > b && b + c > a;
}
// Формула Герона для площади треугольника
function triangleAreaHeron(a, b, c) {
const p = (a + b + c) / 2;
return Math.sqrt(p * (p - a) * (p - b) * (p - c));
}
// Преобразование градусов в радианы
function toRadians(degrees) {
return degrees * Math.PI / 180;
}

JavaScript: Функции для круга

function calculateCircle() {
const r = parseFloat(document.getElementById('circle-r').value);
const errorElement = document.getElementById('circle-error');
if (isNaN(r) || r <= 0) {
errorElement.textContent = 'Ошибка: радиус должен быть положительным числом';
return;
}
errorElement.textContent = '';
const perimeter = 2 * Math.PI * r;
const area = Math.PI * r * r;
const diameter = 2 * r;
document.getElementById('circle-perimeter').textContent = perimeter.toFixed(2);
document.getElementById('circle-area').textContent = area.toFixed(2);
document.getElementById('circle-diameter').textContent = diameter.toFixed(2);
}

JavaScript: Функции для треугольника

function calculateTriangle() {
const a = parseFloat(document.getElementById('triangle-a').value);
const b = parseFloat(document.getElementById('triangle-b').value);
const c = parseFloat(document.getElementById('triangle-c').value);
const errorElement = document.getElementById('triangle-error');
if (isNaN(a) || isNaN(b) || isNaN(c) || a <= 0 || b <= 0 || c <= 0) {
errorElement.textContent = 'Ошибка: все стороны должны быть положительными числами';
return;
}
const isValid = isValidTriangle(a, b, c);
const perimeter = a + b + c;
let area = 0;
if (isValid) {
area = triangleAreaHeron(a, b, c);
errorElement.textContent = '';
} else {
errorElement.textContent = 'Треугольник не существует: нарушено неравенство треугольника';
}
document.getElementById('triangle-perimeter').textContent = perimeter.toFixed(2);
document.getElementById('triangle-area').textContent = isValid ? area.toFixed(2) : 'Н/Д';
document.getElementById('triangle-exists').textContent = isValid ? 'Да' : 'Нет';
}

JavaScript: Функции для трапеции

function calculateTrapezoid() {
const a = parseFloat(document.getElementById('trapezoid-a').value);
const b = parseFloat(document.getElementById('trapezoid-b').value);
const c = parseFloat(document.getElementById('trapezoid-c').value);
const d = parseFloat(document.getElementById('trapezoid-d').value);
const h = parseFloat(document.getElementById('trapezoid-h').value);
const errorElement = document.getElementById('trapezoid-error');
if ([a, b, c, d, h].some(val => isNaN(val) || val <= 0)) {
errorElement.textContent = 'Ошибка: все значения должны быть положительными числами';
return;
}
errorElement.textContent = '';
const perimeter = a + b + c + d;
const areaBases = ((a + b) * h) / 2;
const midline = (a + b) / 2;
const areaMidline = midline * h;
document.getElementById('trapezoid-perimeter').textContent = perimeter.toFixed(2);
document.getElementById('trapezoid-area-bases').textContent = areaBases.toFixed(2);
document.getElementById('trapezoid-area-midline').textContent = areaMidline.toFixed(2);
document.getElementById('trapezoid-midline').textContent = midline.toFixed(2);
}
